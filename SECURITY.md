# Security Policy

## Supported Versions

We support the latest released version. Please verify the issue against
the latest release before reporting.

## Reporting a Vulnerability

If you've found a security issue in this repository's code or one of
its dependencies, please report it privately by emailing
**security@sendsafely.com**.

We'll acknowledge your report and keep you updated as we look into it.
We appreciate responsible disclosure and the work of the security
research community.

---

**Note:** This covers this repository's source code. Vulnerabilities in
the SendSafely platform (`www.sendsafely.com`) go through our
[Bug Bounty Program](https://explore.sendsafely.com/security/bug-bounty/)
via the [Bug Reporting Form](https://explore.sendsafely.com/security/bug-report/).
